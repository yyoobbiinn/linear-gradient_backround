# linear-gradient_backround

### (1) 설명

웹사이트의 버튼을 누를 때마다 색상 두개가 임의로 선택되어 linear-gradient 함수에 넘겨진 후 그레디언트를 만든다.
만들어진 그레디언트를 background 에 적용한다.
range bar 를 통해 linear-gradient 의 각을 조절할 수 있다.
